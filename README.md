A subgraph implementing the [Colony](https://colony.io/) smart contracts on
[The Graph](https://thegraph.com/).

## Building:

To generate the `./generated` folder you first need to run codegen. Then you can build the
project.

```sh
npm run codegen
npm run build
npm run deploy
```
